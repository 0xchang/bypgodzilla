# bypgodzilla
bypgodzilla

哥斯拉WebShell免杀生成

### 警 告

**本工具旨在帮助企业快速定位漏洞修复漏洞,仅限授权安全测试使用!**

**严格遵守《中华人民共和国网络安全法》,禁止未授权非法攻击站点!**

### 使用帮助

* 支持自定义pass和key

* ```bash
  Usage of ./bypgodzilla:
    -k string
          set key
    -o string
          set output file
    -p string
          set password (default "pass")
  ```
